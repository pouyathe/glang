Why G?

G is fast, has a very tiny footprint in your hard disk (only 2.4MB), and features an easy syntax to understand with a powerful package-lib manager called flex.

Built with performance in mind, G offers memory safety without sacrificing speed. Whether you're building system utilities, web servers, or desktop applications, G provides the tools you need with minimal resource consumption.

User Input

    glue gcolor
    glue string

    fcolor __gcolor__:blue
    std.echo "Hi "
    fecho %
    ln

std.end: exit N
