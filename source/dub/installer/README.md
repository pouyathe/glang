## G
Thanks for you want to use G.
Now here is a guide to insatll and use it in your system.

if you want to insatll it to you can use it anywhere, use ```sudo cp glang /usr/bin```
if you just want to do it with script run ```./glang -c ginsatll.g```
> With script also flex installed.

check if it installed ```glang -v``` or just ```glang```

check if flex installed ```flex -v```

if you want to learn G Just go to https://github.com/pouyathe/glang/ and go to folder doc and click on the-g-book-en.md or in persian version.

Also examples live in folder examples which is have some good thing to read and run.
